\doxysubsection{Detailed Description}
A class for analyzing properties of discrete-\/time state space systems. 

This class provides methods for analyzing fundamental system properties including\+:
\begin{DoxyItemize}
\item Controllability\+: ability to drive the system to any desired state
\item Observability\+: ability to determine any initial state from output measurements
\item Stability\+: behavior of the system\textquotesingle{}s free response
\item Stabilizability\+: ability to stabilize unstable modes through feedback
\end{DoxyItemize}

For a discrete-\/time state space system\+: \texorpdfstring{$\ast$}{*} \begin{eqnarray*}
    x[k+1] &=& Ax[k] + Bu[k] \\
    y[k] &=& Cx[k] + Du[k]
    \end{eqnarray*}
